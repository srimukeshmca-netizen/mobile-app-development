


# User Stories - Meditation App

1. As a user, I want to create an account using sign up so that I can access the meditation app features.

2. As a user, I want to log into my account so that I can securely access my personalized data.

3. As a user, I want to view a home screen with popular and daily meditation sessions so that I can choose a meditation easily.

4. As a user, I want to view details of a selected meditation so that I can understand its purpose and benefits.

5. As a user, I want to access different meditation categories so that I can choose the type of meditation I prefer.

6. As a user, I want to change the app theme between light mode and dark mode so that I can use the app comfortably.

7. As a user, I want to log out from the app so that I can securely exit my session.

8. As a user, I want to set daily reminders with date and time so that I can receive meditation notifications regularly.

9. As a user, I want to receive push notifications for reminders so that I am alerted at the scheduled meditation time.
